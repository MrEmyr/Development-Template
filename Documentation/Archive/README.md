# Documentation Archive

This folder stores **superseded versions of documentation**, mirroring the active documentation structure.

Rules (from Draw.io)
--------------------

- **Archive only old files that have been superseded.**
- **Mirror development folders in the archive** so the structure here reflects `Documentation/` (e.g. same subfolder names).

Suggested Practice
------------------

- When replacing an existing document (e.g. updated architecture diagram or plan), move the old version into an appropriate subfolder under `Archive/` using a **relative path**.
